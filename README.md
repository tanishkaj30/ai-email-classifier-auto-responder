# AI Email Classifier & Auto-Responder

## Overview
This project demonstrates an AI-powered email automation workflow built using n8n.
It fetches emails from Gmail, classifies them using an LLM (OpenAI API), sends automated responses for support emails, and stores results in Google Sheets.

## Features
- Gmail email fetching
- AI-based email classification
- Automated email replies
- Google Sheets logging

## Tech Stack
- n8n
- OpenAI API
- Gmail API
- Google Sheets API

## Workflow Architecture
Gmail → n8n → AI Classifier → Decision Logic → Auto Reply → Google Sheets

## Current Status
The live workflow is inactive due to expired API billing.
The full architecture and prompts are documented and can be rebuilt using free LLM alternatives.

- Workflow automation
